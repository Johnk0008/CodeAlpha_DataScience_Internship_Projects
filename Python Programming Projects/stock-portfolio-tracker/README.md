# Stock Portfolio Tracker

A simple Python application to track stock investments with manual price definitions.

## Features
- Add/Remove stocks from portfolio
- Calculate total investment value
- View portfolio summary
- Save portfolio to text file
- Save portfolio to CSV file
- Pre-defined stock prices
